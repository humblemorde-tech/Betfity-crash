# 💥 BETFITY CRASH

A casino-style crash game simulation platform.

## 🎮 Features
- Crash game (real-time)
- Wallet system (KES rules)
- Slots (optional extension)
- Fake deposits & withdrawals

## 💰 Rules
- Minimum deposit: 50 KES
- Minimum withdrawal: 100 KES

## ⚠️ Important
This is a simulation project for learning.
No real money or payment systems included.

## 🚀 Run
Backend:
npm install
node server.js

Frontend:
npm install
npm start
